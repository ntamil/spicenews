# SpiceNews App

*Required PHP version 7+

*No database connection needed

*Bootstrap library used

Artisan cmd to run

Home - http://localhost:8000/

Topic Search - http://localhost:8000/topicSearch

Date Search - http://localhost:8000/dateSearch
